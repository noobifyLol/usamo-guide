## Introduction to Imaginary Numbers

When we learnt of squares and square roots in 5th, we surely would have wondered what is the square root of the negative numbers. If your teacher told you that those dont exist, He/She is both right and wrong at the same time! How? We shall answer that question here.

Solving the equation $x^2 + 1 = 0$ gives us x = $\pm \sqrt{-1}$
Now, what is this $\sqrt{-1}$? That is called $i$ (iota - pronounced as "eye-OH-tuh")

If we multiply $i$ by a real number like 2 or $\pi$, we get a number like $2 i$ or $\pi i$, there is no way to simplify this product. Numbers like this,
formed by multiplying $i$ by a real, are called pure imaginary numbers

If we multiply i by itself, we get $\sqrt{-1} * \sqrt{-1} = (\sqrt{-1}) ^ 2 = - 1$ as we would expect. But notice that if we try to combine the radicals and write $\sqrt{-1} * \sqrt{-1} = \sqrt{(- 1) ^ 2} = 1$ we will get the wrong answer. Manipulations like this are forbidden.

$\textbf{Complex number:}$ A number of the form $z = a + bi$ where $a,b \in \mathbb{R}$ and $i^2 = -1$.
(Pro Tip: This includes all Real Numbers and Imaginary Numbers)

## Algebra of Complex Numbers

$\textbf{Real/Imag parts:}$ 
For $z=a+bi$, $\Re(z)=a$ and $\Im(z)=b$ (the imaginary part excludes the factor $i$).

$\textbf{Addition/Subtraction:}$

 $ (a+bi) \pm (c+di) = (a\pm c) + (b\pm d)i.$

$\textbf{Multiplication:}$ (Just Simple Cross-Multiplication)

 $(a+bi)(c+di) = (ac-bd) + (ad+bc)i.$ 

Note: Divison will be covered later in this page, as it requires some more concepts.

## Solving Linear Equations in Complex Numbers
Recall that two real numbers are equal if and only if they are the same number. For complex numbers, equality is stricter; both the real and imaginary parts must match separately. 

$\textbf{Equating Parts:}$ If $z = w$, then $\Re(z) = \Re(w)$ and $\Im(z) = \Im(w)$.

In other words, a single complex equation is secretly **two** real equations in disguise! This is key for solving complex equations elegantly.

So the strategy is simple: 
- Simplify both sides into the form $a + bi$
- Equate the real parts 
- Equate the imaginary parts
- Solve the resulting pair of real equations

### Example :

Q: Find real $x, y$ such that $(x + yi)(2 + 3i) = 1 + 13i$ $\\$
A: First, expand the left side using our multiplication rule:

$$(x+yi)(2+3i) = (2x - 3y) + (3x + 2y)i$$

Now equating parts with the right side $1 + 13i$:

$$2x - 3y = 1 \quad \text{and} \quad 3x + 2y = 13$$

Multiply the first equation by $2$ and the second by $3$:

$$4x - 6y = 2 \quad \text{and} \quad 9x + 6y = 39$$

Adding them: $13x = 41 \implies x = \frac{41}{13}$ and similarly $y = \frac{3(\frac{41}{13}) - 13}{2}$

For anyone who has handled vectors before, this might seem very similar, why? Because complex numbers are represented as vectors!! We will cover more about this in the next unit. 

## Conjugates of Complex numbers
We have learnt in surds that the conjugate of $\sqrt{a}+\sqrt{b}$ is $\sqrt{a}-\sqrt{b}$ since multiplying them gives $a-b$. Similarly for complex numbers, conjugate of $z = a + bi$ is $\bar{z} = a - bi$, This is as when we multiply them, we get: $\\
z*\bar{z} = (a + bi)(a-bi) = a^2 -abi + abi -b^2 i^2 = a^2  + b^2
$

We also note here that $ z + \bar{z} = 2\Re(z) \text{ and } z - \bar{z} = 2\Re(z)$ this is a key result that would be used often when solving to convert complex equations into their real form. 

## Division of Complex Numbers

Recall that dividing by a number is the same as multiplying by its reciprocal. So, $\frac{1}{a+bi}$ is what we need to make sense of first. Now, the trouble is that we don't want an imaginary number sitting in the denominator (it's messy and hard to work with!) 

How do we fix this? We use the same trick we use with surds! Multiply both the numerator and denominator by the conjugate of the denominator.

$$\frac{1}{a+bi} = \frac{1}{a+bi} \cdot \frac{a-bi}{a-bi} = \frac{a-bi}{a^2+b^2}$$

Notice that the denominator $a^2 + b^2$ is now a **real** number. (This is an very important result. $z \cdot \bar{z}$ always gives us a real number.)

For the general case, dividing $z_1 = a+bi$ by $z_2 = c+di$:

$$\frac{a+bi}{c+di} = \frac{a+bi}{c+di} \cdot \frac{c-di}{c-di} = \frac{(a+bi)(c-di)}{c^2+d^2}$$

Expanding the numerator using our multiplication rule:

$$(a+bi)(c-di) = (ac+bd) + (bc-ad)i$$

Hence, the final result is

$$\frac{a+bi}{c+di} = \frac{(ac+bd)}{c^2+d^2} + \frac{(bc-ad)}{c^2+d^2}i$$


### Example:

Q: Compute $\dfrac{3+2i}{1-4i}$ $\\$
A: Multiply top and bottom by the conjugate of the denominator, which is $1+4i$:

$$\frac{3+2i}{1-4i} \cdot \frac{1+4i}{1+4i} = \frac{(3+2i)(1+4i)}{1^2+4^2}$$

Expanding the numerator: $(3+2i)(1+4i) = (3 - 8) + (12 + 2)i = -5 + 14i$

And the denominator: $1 + 16 = 17$

$$\therefore \quad \frac{3+2i}{1-4i} = \frac{-5+14i}{17} = -\frac{5}{17} + \frac{14}{17}i$$


## Recap:
- $z = a + bi$ with $i^2 = -1$.
- Powers of $i$ repeat every $4$.
- Real part is $a$, imaginary part is $b$.
- $(a+bi) \pm (c+di) = (a\pm c) + (b\pm d)i$.
- $(a+bi)(c+di) = (ac-bd) + (ad+bc)i$.
- If $z = a+bi$, then $\overline{z} = a-bi$.
- Solve equations by isolating $z$ and rationalizing with conjugates.
- $z\overline{z} = a^2 + b^2 = |z|^2$.
- Use conjugates to rationalize denominators.

## Worked Examples

### Example 1

Compute $i^{2023}$.

Powers of $i$ cycle with period $4$. Since $2023 \equiv 3 \pmod 4$,
$i^{2023} = i^3 = -i$.

### Example 2

If $z + 6i = iz$, find $z$.

Rearrange: $z - iz = -6i$, so $z(1-i) = -6i$.
Then
$z = \frac{-6i}{1-i} = \frac{-6i(1+i)}{(1-i)(1+i)} = \frac{6-6i}{2} = 3-3i$.

### Example 3

Let $z + \overline{z} = 6$ and $z\overline{z} = 13$. Find $z$.

Write $z = a+bi$. Then $2a=6$ so $a=3$. Also $a^2+b^2=13$ so $b^2=4$ and
$b=\pm 2$. Thus $z = 3 \pm 2i$.


## Strategy Checklist

- Reduce powers of $i$ modulo $4$.
- Keep real and imaginary parts separate and equate real and imaginary parts when solving equations.
- Rationalize denominators, use the conjugate to eliminate $i$ in denominators.
- Replace $z$ with $a+bi$ to solve real equations.
- Use $|z|^2$ to relate real and imaginary parts.

## Common Pitfalls

- Treating the imaginary part as $bi$ instead of $b$.
- Dropping the $i^2=-1$ sign change.
- Mixing real and imaginary parts after expansion.
- Mixing up $z+\overline{z}$ with $2|z|$.
- Forgetting that $\overline{z}$ changes the sign of the imaginary part only.
- Dropping the denominator after multiplying by the conjugate. (Denominator is $a^2 + b^2$)
