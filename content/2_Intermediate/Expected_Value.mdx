## Overview
A *random variable* is defined as a number whose value is determined by a random process, like a die roll. The *expected value* of a random variable is a fancy way of saying its average value.

Let $P(e)$ be the probability of event $e$ happening. If we have a random variable $X$ with outcomes $x_1, x_2, \ldots, x_n$ occurring with probabilities $p_1, p_2, \ldots, p_n$, then the expected value is the weighted average

$$E[X] = x_1p_1+x_2p_2+\cdot\cdot\cdot+x_np_n.$$

From this definition we clearly have:

1. $E[X+c]=E[X]+c$
2. $E[cX] = cE[X]$

for some constant $c$.

There are also two other very important, but a bit less obvious, facts about expected value.
1. If $X, Y$ are two *independent* random variables, then $E[XY] = E[X]E[Y]$.
2. *Linearity of Expectation*: If $X,Y$ are two random variables, then $E[X+Y]=E[X]+E[Y]$. This is true *regardless* of whether or not $X,Y$ are independent. **This is arguably the most important property of expected value.**
### A Simple Example
In a dice roll, the possible outcomes are $1,2,3,4,5,6$ and the probability of each is $\frac{1}{6}$, so the expected value is

$$\frac{1}{6}\cdot1+\frac{1}{6}\cdot2+\frac{1}{6}\cdot3+\frac{1}{6}\cdot4+\frac{1}{6}\cdot5+\frac{1}{6}\cdot6 = 3.5$$
The expected sum of two dice rolls is $3.5+3.5=7$.
The expected product of two dice rolls is $3.5\times3.5=12.25$, as the dice rolls are independent.

## Worked Example #1
Suppose there are $n$ people, each of whom has a name tag. The name tags are shuffled, and each person is randomly given a name tag. What is the expected number of people who received their own name tag?
> ### Solution
> Label the people $1,2,...,n$. The trick here is to define the *indicator variable* $X_i$ to be 0 if person $i$ does not get his name tag and 1 if person $i$ does get his name tag. Notice that $X_1+X_2+\cdot\cdot\cdot+X_n$ is just the number of people who got their own name tag.
>
> > Using indicator variables which take on a value of 0 or 1 is a common trick, since it allows us to transform counting problems into expected value problems.
>
> $E[X_i]=\frac{1}{n}$, as each person has a $\frac{1}{n}$ probability of getting their own name tag. Hence the expected number of people who got their own name tag is $E[X_1+X_2+\cdot\cdot\cdot+X_n]=E[X_1]+E[X_2]+\cdot\cdot\cdot+E[X_n]=n\cdot\frac{1}{n}=\boxed{1}$.

## Worked Example #2 (2021 Fall AMC 10B #16)
Five balls are arranged around a circle. Chris chooses two adjacent balls at random and interchanges them. Then Silva does the same, with her choice of adjacent balls to interchange being independent of Chris's. What is the expected number of balls that occupy their original positions after these two successive transpositions?
> ### Solution
> Once again we use indicator variables. Label the balls $1,2,3,4,5$ and define $X_i$ to be 1 if ball $i$ remains in its original position after the transpositions and 0 if it does not. Then the number of balls that are in their original positions is $X_1+X_2+X_3+X_4+X_5$.
>
> There is a $(\frac{3}{5})^2$ chance that a given ball does not get swapped and a $\frac{2}{5}\cdot\frac{1}{5}$ chance the ball returns to its original position after being swapped by both Chris and Silva. So $E[X_i]=(\frac{3}{5})^2+\frac{2}{5}\cdot\frac{1}{5}=\frac{11}{25}$. Hence the expected number of balls that occupy their original positions is $E[X_1+X_2+X_3+X_4+X_5]=E[X_1]+E[X_2]+E[X_3]+E[X_4]+E[X_5]=5\cdot\frac{11}{25}=\boxed{2.2}$.

## Worked Example #3 (2021 Fall AMC 12B #23)
What is the average number of pairs of consecutive integers in a randomly selected subset of $5$ distinct integers chosen from the set $\{ 1, 2, 3, …, 30\}$? (For example the set $\{1, 17, 18, 19, 30\}$ has $2$ pairs of consecutive integers.)
> ### Solution
> There are $29$ pairs of consecutive integers, namely $p_1 = \{1,2\}, p_2=\{2,3\},...,p_{29}=\{29,30\}$. Define the indicator variable $X_i$ be equal to 0 if pair $p_i$ is not included the subset and 1 if it is.
>
> For any given pair $p_i$, there are $\binom{28}{3}$ possible subsets that include $p_i$ out of a possible $\binom{30}{5}$ subsets. So the probability that $p_i$ is included in the subset is $\frac{\binom{28}{3}}{\binom{30}{5}}=\frac{1}{29}\cdot\frac{2}{3}$. Hence $E[X_i]=\frac{1}{29}\cdot\frac{2}{3}$
>
> Then $E[X_1+X_2+\cdot\cdot\cdot+X_{29}]=E[X_1]+E[X_2]+\cdot\cdot\cdot+E[X_{29}]=29\cdot\frac{1}{29}\cdot\frac{2}{3}=\boxed{\frac{2}{3}}$
>
> >Expected value can be an extremely powerful tool for simplifying counting problems, as you can see by taking a look at the [solutions to this problem that don't use expected value](https://artofproblemsolving.com/wiki/index.php/2021_Fall_AMC_12B_Problems/Problem_23). Linearity of expectation allows us to break apart a complicated structure into simple parts without worrying about how they will affect each other.
