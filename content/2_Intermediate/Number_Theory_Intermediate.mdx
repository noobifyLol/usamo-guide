## Overview

Intermediate number theory emphasizes modular arithmetic, gcd arguments, and
careful factoring.

## Key Ideas


## Core Skills

### Reduce Modulo a Convenient Base

Pick moduli that simplify the expression (like 2, 3, 4, 5, 8, 9, 11).

### Use the Euclidean Algorithm

Compute gcds quickly to test solvability of linear Diophantine equations.

### Combine Congruences

Use the Chinese Remainder Theorem for compatible mod conditions, even in
simple two-modulus cases.

## Worked Example

Solve $7x\equiv 1\pmod{10}$.

Check small residues: $7\cdot 3=21\equiv 1\pmod{10}$, so $x\equiv 3\pmod{10}$.

## More Examples

### Example 1: Linear Diophantine

Determine if $6x+15y=9$ has integer solutions.

$\gcd(6,15)=3$ divides $9$, so solutions exist.

### Example 2: Mod Elimination

Can $x^2\equiv 3\pmod 4$ happen?

Squares are $0$ or $1$ mod $4$, so no.

### Example 3: CRT Quick Solve

Find $x$ with $x\equiv 2\pmod 3$ and $x\equiv 1\pmod 4$.

Check $x=5$, which satisfies both; so $x\equiv 5\pmod{12}$.

### Example 4: Interesting Sequence (hint: requires the Euclidean Algorithm!)

Let $a_1 = 10^9+7$ and $a_2 = 10^9+9$. For $n \ge 3$, we will let $a_n$ be the remainder of $a_{n-2} \div a_{n-1}$. Now, given that $X \equiv 2 \pmod{a_3}$ and $X \equiv 3 \pmod{a_4}$, your job is to compute the smallest positive solution for $X$.

We can start off by first using the Euclidean algorithm which gives $a_3 = 10^9+7$ and $a_4 = 2$. Then, by CRT (Chinese remainder theorem), we get that $X \equiv 2 \pmod{10^9+7}$ and $X \equiv 3 \equiv 1 \pmod 2$. Lastly, equating $X = 2 + (10^9+7) = 10^9+9$ fulfills both conditions well.

### Example 5: Exponent On Top Of Exponent On Top of Exponent...

Your job is to find the last two digits of $7^{\left( 7^{\left( 7^7 \right)} \right)}$.

We first compute $X = 7^E$ modulo $4$ and $25$.Modulo $4$, $7^E \equiv (-1)^{\text{odd}} \equiv 3 \pmod 4$.Modulo $25$, $\phi(25)=20$, and so we will need $E = 7^{(7^7)} \pmod{20}$. We can split the modulus $20$: $E \equiv 3 \pmod 4$ and $E \equiv 2^3 \equiv 3 \pmod 5$, so $E \equiv 3 \pmod{20}$.Thus, $7^3 = 343 \equiv 18 \pmod{25}$.Lastly, we can combine $X \equiv 3 \pmod 4$ and $X \equiv 18 \pmod{25}$ via CRT (Chinese remainder theorem), which gives that $X \equiv 43 \pmod{100}$.

### Example 6: Hidden Diophantine Equation!

Your goal is to compute the number of ordered pairs of positive integers $(x, y)$ satisfying $\frac{4}{x} + \frac{2}{y} = \frac{1}{6}$.

First, we can clear the denominators, which gives us $24y + 12x = xy$, and can simply rearrange to $xy - 12x - 24y = 0$. Next, we can use Simon's Favorite Factoring Trick which results in $(x - 24)(y - 12) = 288$. Furthermore, because $288 = 2^5 \times 3^2$, it thus has $(5+1)(2+1) = 18$ positive divisors.Now, also, each positive divisor pair produces a unique positive integer solution for $(x,y)$, but on the other hand, negative divisor pairs force $x \le 0$ or $y \le 0$. From this, we get that there are exactly $18$ solutions.

## Strategy Checklist

- Choose a modulus that simplifies the expression.
- Check gcd conditions for linear equations.
- Verify solutions by substitution.

## Practice Problems

- Using a modulus that does not provide new information.
- Forgetting to check gcd divisibility.
- Assuming a congruence has a solution without testing residues.
